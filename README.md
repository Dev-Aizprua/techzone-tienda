# TechZone - Tienda de Tecnología
Tienda e-commerce de productos tecnológicos con integración a Google Sheets.

## Características
- Catálogo de productos
- Carrito de compras
- Sistema de pedidos
- Emails automáticos
- Control de inventario

## Tecnologías
- HTML/CSS/JavaScript
- Google Sheets API
- Vercel (hosting)
